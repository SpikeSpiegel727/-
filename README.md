# Система управления задачами с приоритетами

## Описание

Программа позволяет пользователю вести список задач. Для каждой задачи указывается название, приоритет, время выполнения и дедлайн.

## Структуры данных

- **Стек** — хранит историю действий (добавление, удаление, изменение приоритета) для возможности отмены.
- **Очередь** — хранит задачи, поставленные на исполнение.
- **Бинарное дерево поиска (BST)** — строится на основе дедлайнов. Позволяет выводить задачи в порядке возрастания дедлайна (рекурсивный симметричный обход), а также быстро находить задачу с самым ранним и самым поздним сроком.

## Запуск

```
python main.py
```

## Функциональность

1. Добавить задачу
2. Удалить задачу
3. Изменить приоритет задачи
4. Показать все задачи
5. Отменить последнее действие
6. Добавить задачу в очередь на исполнение
7. Взять задачу из очереди
8. Показать задачи по дедлайну (обход дерева)
9. Найти задачу с ближайшим дедлайном
10. Найти задачу с самым поздним дедлайном
